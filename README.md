# NPM - Art

> Drop a random ASCII pattern in your terminal.

25+ tiling ASCII patterns scraped from [ascii-patterns/gallery](https://www.asciiart.eu/ascii-patterns/gallery), printed with ANSI color — zero dependencies.

```
  ✦ Honeycomb
  ──────────────────────────────────────────────────

  / \ / \ / \ / \ / \ / \
  \_\_/ \\\_\_/ \\\_\_/ \\\_\_/ \\\_\_/ \\\_\_/
  \ / \ / \ / \ / \ / \
  \\\_\_/ \\\_\_/ \\\_\_/ \\\_\_/ \\\_\_/ \\\_\_/

  ──────────────────────────────────────────────────
  patterndrop — ascii-patterns/gallery
```

## Usage

```bash
# Random pattern
npx patterndrop

# List all pattern names
npx patterndrop --list

# Print a specific pattern
npx patterndrop --name "Honeycomb"
```
